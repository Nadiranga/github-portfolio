# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.

This project is designed for students, developers, data scientists, and software engineers who want to showcase their:

- Projects
- Skills
- Education
- Resume
- Social Links
- Contact Information

## ⚡ How to Use This Portfolio

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Nadiranga/github-portfolio.git
```

---

### 2️⃣ Open the Project

Open the folder in:

- VS Code
- Sublime Text
- Any Code Editor

---

### 3️⃣ Customize Your Information

Edit:

```bash
index.html
```

Replace:

- Name
- About Section
- Projects
- Skills
- Education
- Social Links
- Resume
- Images

---

### 4️⃣ Add Your Images

Place your images inside:

```bash
/images
```

---

### 5️⃣ Run the Project

Simply open:

```bash
index.html
```

in your browser.

---

## 🌍 Deploy on GitHub Pages

1. Push your code to GitHub
2. Go to:

```bash
Settings → Pages
```

3. Select:

```bash
Deploy from Branch → main
```

4. Save

Your portfolio will be live at:

```bash
https://yourusername.github.io/repository-name/
```
